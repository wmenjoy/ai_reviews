# 对齐（ALIGNMENT）

## 原始需求
- 开发一个 VSCode 插件，提供简单的记事本（Notepad）功能。
- 在当前项目（工作区）与用户目录（全局）维护有用提示语或其他信息。
- 在 VSCode 输入时为用户提供提示与帮助（补全与内联）。

## 任务范围与边界
- 范围：侧边栏管理笔记、命令快捷操作、Completion/Inline 提示、工作区与全局存储、导入导出。
- 不包含：云同步、团队权限控制、富文本编辑器（首版使用 TreeView + 输入框，后续可升级 Webview）。

## 项目理解
-- 插件提供两类数据源：工作区 `.vscode/notepad.json` 与全局 `globalStorage/notepad.json`。
-- 输入时根据前缀、语言、路径、置顶与频次等进行排名，展示候选与幽灵文本。
-- 通过侧边栏进行 CRUD 与搜索过滤；状态栏提供快速插入入口。

## 关键假设与决策
-- 默认优先使用工作区文件，便于团队共享；可通过配置关闭。
-- 内容支持 VSCode Snippet 语法，插入时保留占位符。
-- 所有数据本地存储，不进行网络传输。

## 待确认点（已采用默认）
-- 笔记编辑首版采用原生输入控件；如需富编辑器，将在后续采用 WebviewView。
-- Inline 幽灵文本默认启用，支持全部语言；可在设置中限制。
